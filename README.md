### This project demonstrates how to:
* Install Lua 5.1/5.2/5.3
* Install LuaRocks
* make and install Lua module containing C and Lua parts
* Run test

It is based on @moteus [lua-travis-example](https://github.com/moteus/lua-travis-example).
